### Founder-Led Demo Script

**Structure** (15 minutes):

1. **Problem** (2 min)
   - "LLM costs are eating into margins"
   - Show their specific pain point

2. **Solution** (3 min)
   - Show Cost Melt dashboard
   - Explain features (routing, caching, compression)
   - Highlight zero code changes

3. **Integration** (5 min)
   - Live code demo
   - Add Cost Melt to their app
   - Show before/after

4. **Results** (3 min)
   - Show savings, analytics, ROI
   - Real numbers from similar companies

5. **Q&A** (2 min)
   - Answer questions
   - Address objections
   - Next steps

**Goal**: Close deal or schedule follow-up
